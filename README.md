# Calculator

A modern, responsive web-based calculator with a sleek dark UI and smooth animations.

## Features

- **Basic Arithmetic Operations**: Add, subtract, multiply, and divide numbers
- **Advanced Functions**: Modulo (%) operation support
- **Easy-to-Use Controls**:
  - **AC**: Clear all inputs and reset the calculator
  - **DEL**: Delete the last entered digit
  - **=**: Calculate the result
- **Modern Design**: Dark gradient background with smooth button animations
- **Responsive Layout**: Optimized for various screen sizes
- **Real-time Display**: Live input feedback as you type

## Project Structure

```
├── index.html       # HTML structure of the calculator
├── script.js        # JavaScript logic for calculator operations
├── style.css        # Styling and animations
└── readme.md        # Project documentation
```

## Getting Started

### Prerequisites

No installation or dependencies required. Just a modern web browser.

### Installation

1. Clone the repository or download the files
2. Open `index.html` in your web browser
3. Start calculating!

## Usage

1. **Enter Numbers**: Click number buttons (0-9) to input digits
2. **Select Operation**: Click an operator button (+, -, *, /, %)
3. **View Result**: Press the equals (=) button to calculate
4. **Clear**: Use AC to clear all entries or DEL to remove the last digit

### Example Operations

- `10 + 5 =` → `15`
- `20 - 8 =` → `12`
- `6 * 4 =` → `24`
- `15 / 3 =` → `5`
- `25 % 7 =` → `4`
